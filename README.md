# procon32 Let's AIdea React

## 使い方

```
git clone https://github.com/NIT-Anan-Procon/procon32_Lets_AIdea_react.git
npm install
npm run dev
```

## コマンドについて

- `npm run dev`
  - 開発サーバーを起動します。
- `npm run fix`
  - `format` と `lint` を 1 つのコマンドで行います
- `npm run format`
  - Prettier を適用します。
- `npm run lint`
  - ES Lint を適用します。
- `npm run build`
  - ビルドファイルを作成します。
- `npm run serve`
  - serve を使ってサーバーを起動します。
