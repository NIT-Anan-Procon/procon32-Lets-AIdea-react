# procon32 Let's AIdea React

## 使い方

```
git clone https://github.com/NIT-Anan-Procon/procon32_Lets_AIdea_react.git
npm install
npm run dev
```

## コマンドについて
- `npm run dev`
  - 開発サーバーを起動します。
- `npm run fix`
  - `format` と `lint` を1つのコマンドで行います
- `npm run format`
  - Prettierを適用します。
- `npm run lint`
  - ES Lintを適用します。
- `npm run build`
  - ビルドファイルを作成します。
- `npm run serve`
  - serveを使ってサーバーを起動します。
